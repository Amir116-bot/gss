# نظام طلبات العمل

نظام ويب متكامل لتقديم وإدارة طلبات العمل مع قاعدة بيانات MySQL.

## المميزات

- ✅ نموذج تقديم طلبات العمل بتصميم جميل
- ✅ التحقق من صحة البيانات
- ✅ تشفير كلمات المرور
- ✅ عرض قائمة الطلبات المقدمة
- ✅ إحصائيات الطلبات
- ✅ تصميم متجاوب
- ✅ دعم اللغة العربية

## متطلبات النظام

- PHP 7.4 أو أحدث
- MySQL 5.7 أو أحدث
- خادم ويب (Apache/Nginx)
- متصفح ويب حديث

## خطوات الإعداد

### 1. إعداد قاعدة البيانات

1. افتح phpMyAdmin أو أي أداة إدارة MySQL
2. قم بتشغيل محتوى ملف `database.sql` لإنشاء قاعدة البيانات والجدول

### 2. تكوين الاتصال

1. افتح ملف `config.php`
2. عدّل إعدادات قاعدة البيانات حسب بيئتك:

```php
$host = 'localhost';        // عنوان الخادم
$dbname = 'job_applications'; // اسم قاعدة البيانات
$username = 'root';         // اسم المستخدم
$password = '';             // كلمة المرور
```

### 3. رفع الملفات

1. انسخ جميع الملفات إلى مجلد خادم الويب
2. تأكد من أن المجلد قابل للكتابة (للصور إذا كنت تريد إضافة ميزة رفع الملفات)

### 4. اختبار النظام

1. افتح `listmogin.html` في المتصفح
2. جرب تقديم طلب عمل
3. تحقق من `view_applications.php` لعرض الطلبات

## هيكل الملفات

```
├── listmogin.html          # صفحة النموذج الرئيسية
├── liststyle.css           # ملف التصميم
├── config.php              # إعدادات قاعدة البيانات
├── process_form.php        # معالجة النموذج
├── view_applications.php   # عرض الطلبات
├── database.sql            # ملف إنشاء قاعدة البيانات
└── README.md               # هذا الملف
```

## الأمان

- ✅ تشفير كلمات المرور باستخدام `password_hash()`
- ✅ حماية من SQL Injection باستخدام Prepared Statements
- ✅ تنظيف وتحقق من البيانات المدخلة
- ✅ حماية من XSS باستخدام `htmlspecialchars()`

## تخصيص النظام

### تغيير الألوان

يمكنك تعديل الألوان في ملف `liststyle.css`:

```css
:root {
    --primary-color: #312a7c;  /* اللون الأساسي */
    --white-color: #fff;       /* اللون الأبيض */
    --black-color: #000;       /* اللون الأسود */
}
```

### إضافة حقول جديدة

1. أضف الحقل في `listmogin.html`
2. أضف التحقق في `process_form.php`
3. أضف العمود في `database.sql`
4. أضف العرض في `view_applications.php`

## استكشاف الأخطاء

### مشاكل الاتصال بقاعدة البيانات

1. تأكد من تشغيل خدمة MySQL
2. تحقق من إعدادات الاتصال في `config.php`
3. تأكد من وجود قاعدة البيانات

### مشاكل في النموذج

1. تحقق من أسماء الحقول في HTML
2. تأكد من إعدادات PHP
3. راجع سجلات الأخطاء في الخادم

## الدعم

إذا واجهت أي مشاكل، يرجى التحقق من:

3. إعدادات قاعدة البيانات
